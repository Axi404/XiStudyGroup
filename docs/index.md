# 关于我们

汐学组是西安交通大学人工智能专业的一些学生自发组织的类学辅组织，采用扁平化管理，成果开源。

本学组中绝大多数资料整理会发布在AI学组中，并且进行维护。

本网页主要用于《人工智能新生指南》的发布。
